HTML

O HTML (HyperText Markup Language), ou Linguagem de Marcação de HiperTexto, é um formato de arquivo em texto com marcações dando a este uma estrutura. As marcações são definidas utilizando tags que representam alguma informação da página Web. Todas as tags necessitam que sejam iniciadas com suas marcações entre "<" e ">" e fechadas da mesma forma, mas com o acréscimo de uma barra ("/") antes da tag.

Para que o navegador entenda que o documento que ele está abrindo é um arquivo HTML, a página precisa de utilizar a tag "<html></html>". Dentro desta tag, será construída sua página.

Dentro da tag de html, a página possui um cabeçalho definido pela tag "<head></head>". Dentro desta tag, são definidos o título da página, que aparece na aba do seu navegador, e os metadados presentes na página. Estes metadados não são mostrados. Estes metadados costumam ser links para documentos de estilo da página, conjunto de caracteres, scripts, entre outros.

Após a tag head, ainda dentro da tag de html, está presente a tag "<body></body>" e é dentro dessa tag que estará todo o corpo de sua página.

Depois disso tudo, sabemos que um documento html tem essa cara:

<html>
<head>
<title>Título da página</title>
</head>
<body>
</body>
</html>
