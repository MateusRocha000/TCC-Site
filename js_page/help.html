JavaScript

O JavaScript é uma linguagem de programação que realiza o controle dos elementos de HTML e CSS, manipulando o comportamento dos mesmos. Um exemplo, para deixar mais claro, é o de um submenu. A ação dele aparecer ao se clicar na opção do menu principal é feita em JavaScript. Muitas coisas podem acontecer na página, desde o momento em que é carregada, passando por eventos que ocorrem na página e indo até mesmo ao momento em que a página é fechada. Tudo isso pode ser controlado pela linguagem JavaScript.

Em seu código são necessários alguns elementos básicos para fazer o controle da página.

Caso seja necessário guardar uma informação, você utiliza variáveis com as palavras reservadas "var" ou "let". A primeira é usada quando a variável possui um escopo global, ou seja, ela pode ser usada fora do local onde foi declarada, enquanto a segunda é usada para variáveis com escopo local, invisível para acessos por fora.

Muitas das vezes essa variável é utilizada para ser associada à um elemento HTML para que seja possível ser trabalhado em cima do mesmo. Se o elemento possui uma class, o elemento pode ser pego pelo nome da classe, com a função "getElementByClassName", enquanto para id ele é pego pelo id: getElementById. O nome do elemento é colocado entre aspas, simples ou duplas, como parâmetro da função.

Ainda há uma outra forma: o "querySelector". Com ele você pode acessar elementos HTML pelo nome de sua tag também (inclusive é o mais recomendado a se utilizar). Mas, é preciso inserir antes dos nomes "#" para elementos pegos por id e "." para class. No caso de nomes das tags, não é necessário inserir algum símbolo.
