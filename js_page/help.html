JavaScript

O JavaScript é uma linguagem de programação que realiza o controle dos elementos de HTML e CSS, manipulando o comportamento dos mesmos. Muitas tarefas podem ser feitas em JavaScript, como um submenu aparecer ao clicar no principal, uma janela aparecer ao carregar uma página ou o reconhecimento do código que você está digitando nas atividades.

Em seu código são necessários alguns elementos básicos para fazer o controle da página.

Caso queria descobrir quais comandos são, acesse: JavaScript.
