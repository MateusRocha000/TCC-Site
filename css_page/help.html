CSS

O CSS (Cascading StyleSheet), ou Folha de Estilo em Cascata, é um formato de arquivo utilizado para dar estilo aos elementos em uma página HTML. Com ela é possível alterar a cor de fundo, tamanho da fonte, posição dos elementos e muitas outras coisas.

Para alterar as propriedades de um elemento, dando um estilo ao mesmo, apenas é necessário que você coloque o nome do elemento sucedido de chaves.

Um exemplo: caso queira alterar o estilo de um parágrafo, basta colocar no seu arquivo (".css") o nome do elemento, a tag sem os símbolos "<" e ">", ou seja "p" com as chaves e a propriedades dentro. Caso queira alterar o tamanho da fonte do parágrafo coloque:

p {
font-size: 40px;
}

Assim, o tamanho do texto, ou da fonte, do parágrafo será alterado para 40 pixels.

Há muitas outras propriedades que são possíveis de alterar, como: largura (width), altura (height), cor de fundo (background-color), e muito mais.
